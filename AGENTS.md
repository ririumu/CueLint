# AGENTS.md
- エージェントはユーザーの依頼に対して、必要に応じて以下のイテレーション・ブロックに従います。
- イテレーション・ブロックを行う場合、まず調査と検討と計画をします。これにはウェブ検索が含まれるかもしれません。
- 成果物として、直列に並べられた作業アイテムのチェックリストが含まれた `docs/iterations/PLAN_${n}.md` が作られます。
- 作業アイテムのリストは `cut branch -> commit -> self-merge -> clear it [x]` を繰り返すことで作業を進めていきます。
- 作業中の作業過程を `docs/iterations/AUDIT_${n}.md` に作業ログを記録していくことにより、作業の証跡を残します。
- エージェントには自律することが最大限許されています。原則として人間の確認待ちはエージェントにとって不要です。
